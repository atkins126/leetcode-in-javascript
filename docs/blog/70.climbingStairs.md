# [70] Climbing Stairs

递归写法

```js
var climbStairs = function(n) {
  if (n === 0 || n === 1) return 1;
  return climbStairs(n - 1) + climbStairs(n - 2);
};
```

状态转移方程： dp[n] = dp[n-1] + dp[n-2]

```js
var climbStairs = function(n) {
  const dp = [1, 1];
  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
  }
  return dp[n];
};
```

由于状态转移方程只依赖n-1与n-2，因此可以只用两个变量存储中间值以节省空间复杂度

```js
var climbStairs = function(n) {
  let dp1 = 1;
  let dp2 = 1;
  while (--n) {
    const temp = dp1 + dp2;
    dp2 = dp1;
    dp1 = temp;
  }
  return dp1;
};
```
